專案:架設twitch實況主Minecraft私服

專案簡介

這是一個我自行架設的 Minecraft 伺服器，提供穩定的遊玩體驗(ping約3~4毫秒)，最多支援 40-50 位玩家同時連線。伺服器架設於自家 Ubuntu Server 上，並結合網域名稱（mc.xieserver.com）提供方便的連線方式。

功能說明

• 支援 Java 版 Minecraft 1.20.6

• 白名單制度：有效防止不明玩家進入伺服器破壞環境。

• 自動清除掉落物：每 10 分鐘清除一次無主掉落物，減少伺服器 lag。

• DC 監控 bot自動監控伺服器狀態

技術細節

• 作業系統：Ubuntu Server（安裝於 Proxmox VM 上）

• 伺服器核心：PaperMC 1.20.6

• 網路：使用固定 IP 並透過動態 DNS 服務綁定網域名稱

• Bot 整合：以 Python 撰寫 Discord Bot，自動每 5 分鐘推送伺服器狀態

加入方式

1. 訂閱ugi

2. 加入ugi官方Discord群組

3. 收到Discord訂閱者身分組後填表單申請加入

4. 審核通過後開啟Minecraft並於加入連結打上mc.xieserver.com

心得

這次從零開始架設 Minecraft 伺服器是一段很扎實的實作經驗，不僅學會了伺服器虛擬化與網路設定，也進一步理解如何部署穩定的長期遊玩環境。 途中遇到DNS 設定錯誤、插件版本不相容等問題，都是讓我實際練習問題排查與解決的好機會。透過持續修正與測試，最終打造出穩定的伺服器環境，也讓我更了解系統維運的概念。
